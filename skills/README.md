# SmartTec skills

Machine-readable operating knowledge for this repository, usable as Claude Skills or as
onboarding docs for any contributor (human or agent).

- `design-theme/` — the complete visual design system: tokens, typography, motifs,
  infographic and PDF conventions, motion rules, and don'ts.
- `site-operations/` — stack, deployment workflow, content system, claims policy,
  SEO/GEO doctrine, and hard-won technical gotchas.

Business facts (pricing internals, vendor details, evaluation gates) live in a private
fact base held by the owner and are intentionally not in this public repository.
